# Memories Folder

Place your personal photos and videos in this folder.

## File naming:
- Photos: photo1.jpg, photo2.jpg, etc.
- Videos: video1.mp4, video2.mp4, etc.
- Video thumbnails: video1-thumb.jpg, video2-thumb.jpg, etc.

Update the `/src/data/memories.js` file to match your actual media files with appropriate captions and dates.
